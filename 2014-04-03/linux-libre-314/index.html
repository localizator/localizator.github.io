Вийшло вільне ядро GNU Linux-libre 3.14-gnu.

Date 03 квітень 2014 Tags linux / kernel / libre

Вийшло вільне ядро GNU Linux-libre 3.14-gnu з кодовим ім'ям «Freedom Pi». GNU Linux-libre — вільна версія ядра Linux, яка підходить для використання з операційною системою GNU.

У даному ядрі видалені всі пропрієтарні компоненти. Ядро Linux-libre було спочатку в дистрибутиві gNewSense GNU/Linux. У 2008 році ядро стало частиною проекту, підтримуваним FSF в Латинській Америці, а в 2012 році стало частиною проекту GNU.

Талісманом вільного ядра є Freedo, блакитний пінгвін, який щойно вийшов з душу.

Список змін:

Припинена підтримка aic7xxx_old и cs46xx.
Додана функція reject_firmware_direct в сценарій чистки коду deblob. Пропрієтарні компоненти були видалені з драйверів bnx2x, bt_mrvl, cs46xx.
Припинена підтримка btmtk.
